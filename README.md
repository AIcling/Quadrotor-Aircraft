# 在廖老师的指导和队友的帮助下完成的四轴无人机课设项目

## 代码结构如下:

### *Hardware* :相关传感器外设驱动

### *Lib* :标准固件库

### *Records* :相关图片和文字记录

### *Tools* :相关通讯协议和上位机以及云端椭球拟合代码

### *Start* :与系统初始化相关的启动文件

### *User* :Main函数与中断

### *libc.a libm.a* :用到的静态库

### *其他* :makefile、link、以及启动文件
